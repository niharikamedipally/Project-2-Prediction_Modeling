# 🚢 Titanic Survival Prediction - Machine Learning Project

This project predicts passenger survival on the Titanic using machine learning algorithms.

## 📊 Dataset
Titanic dataset containing passenger details like:
- Age
- Gender
- Class
- Fare
- Embarked location

## ⚙️ Data Preprocessing
- Handled missing values (Age, Embarked)
- Dropped Cabin column
- Converted categorical data into numerical values

## 🤖 Models Used
- Logistic Regression
- Decision Tree Classifier
- Random Forest Classifier

## 📈 Evaluation
- Accuracy Score
- Confusion Matrix
- Model comparison

## 🧰 Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## 🎯 Goal
To compare different machine learning models and understand which performs best in predicting survival.
